# Tic-Tac-Toe-Game
Developed a fully functional 3 x 3 Tic-Tac-Toe game using the Tkinter library for a Graphical User Interface. 
Utilized advanced Python concepts such as Lambda functions for button commands and List Comprehension for efficient grid initialization. 
Implemented core game logic including turn-based play, win-condition detection (8 possible combinations), and draw scenarios.
Integrated messagebox pop-ups to provide real-time game status and winner announcements to the user.
